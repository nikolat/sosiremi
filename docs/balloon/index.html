伺か用バルーン「魔除けの獣面紋」

制作: 月波 清火
公開日: 2023-01-19
動作確認: SSP/2.6.28
------------------------------------------------------------

古代中国をイメージしたバルーンで、
獣面紋(饕餮文とも)や怪獣文をアレンジして取り入れています。

特徴
・大き目(フォントサイズ15、縦長め)
・左右同一デザイン
・スクロール用矢印がやや特殊(右が前への、左が後ろへの矢印)
